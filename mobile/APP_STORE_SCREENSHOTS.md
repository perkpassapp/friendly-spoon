# PerkPass Screenshot Plan

This is the simplest screenshot set to prepare for App Store submission.

## Goal

Show the app as a polished member experience for existing active members.

## Recommended 6-Screen Set

1. Login screen
Caption:
Member access made simple

What to show:
- Welcome screen
- Google sign-in
- Email sign-in

2. Deals screen
Caption:
Browse live local deals

What to show:
- Deal cards
- Day filters
- Category filters

3. Favorites screen
Caption:
Save your go-to spots

What to show:
- Favorited businesses
- Saved deals

4. Redemption code screen
Caption:
Redeem in seconds

What to show:
- Active code
- Countdown
- clear redemption state

5. Redemption history screen
Caption:
Track recent activity

What to show:
- Confirmed and recent redemptions

6. Account screen
Caption:
Billing, support, and account tools

What to show:
- membership details
- billing button
- support access
- legal links

## Capture Notes

- Use the iPhone Simulator for clean captures if possible
- Use the same device size for every screenshot set
- Prefer realistic sample data
- Avoid demo states that say preview unless you have no better option
- Avoid showing any personal email you do not want on store assets
- Use active-member states wherever possible

## Nice-To-Have Extra Screens

- Account deletion flow
- Code confirmed screen
- Live deals now filter state

## Suggested Order In App Store Connect

1. Deals
2. Redemption
3. Favorites
4. History
5. Account
6. Login
