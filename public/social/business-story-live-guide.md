# PerkPass Business Story Motion Template

Use this when a business wants to tell their followers they are live on PerkPass.

## Preview

Start the site:

```bash
npm run dev
```

Open:

```text
http://localhost:3000/social/business-story-live.html
```

Customize the business and deal with URL parameters:

```text
http://localhost:3000/social/business-story-live.html?business=La%20Colombe&deal=$2%20off%20any%20espresso%20drink%20before%2011am
```

## Recommended Story Copy

Post caption:

```text
We are now live on PerkPass.

Find our member-only perk inside the app and show your code at checkout.

Follow @GetPerkPass to learn more.
```

Sticker ideas:

```text
NEW
NOW LIVE
MEMBER PERK
TAP TO JOIN
```

## Export Options

This computer does not currently have `ffmpeg` installed, so the fastest free export workflow is:

1. Open the customized URL in Chrome.
2. Put the browser in full screen.
3. Use QuickTime Player > New Screen Recording.
4. Record one loop of the Story animation.
5. Trim to 6-8 seconds.
6. Upload to Instagram Story and tag `@GetPerkPass`.

If `ffmpeg` is installed later, this can be exported directly to MP4 from the browser animation.
